# All

The `all` CSS property resets _all_ of an element's CSS properties except `unicode-bidi`, `direction`, and custom properties (CSS variables).

Specify `initial`, `inherit`, `unset`, or `revert`.

```css
* {
  all: initial;
  all: inherit;
  all: unset;
  all: revert;
}
```
