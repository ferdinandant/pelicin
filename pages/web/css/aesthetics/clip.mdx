# Clip

Clip CSS properties define a visible portion of an element.

## Clip Properties

### `clip-path`

Defines the shape that determines which part of an element should be visible.

### `clip`

<NoteBox type="danger">
  Deprecated, use the newer <Code str="clip-path" /> property instead.
</NoteBox>

Defines a visible portion of an element. Only applies to absolutely positioned elements
(i.e., those with `position: absolute` or `position: fixed`.)

Specify:

<DocumentationList
  items={[
    {
      name: 'auto',
      description: <>no clipping (default)</>,
    },
    {
      nameSpecial: (
        <>
          <code>
            rect(<var>top</var>, <var>right</var>, <var>bottom</var>, <var>left</var>)
          </code>
        </>
      ),
      description: (
        <>
          e.g. <Code str="rect(0, 175px, 113px, 0)" /> (as offsets from the top/left border);
          defines the rectangle for the clipping
        </>
      ),
    },
  ]}
/>

```css
div {
  clip: rect(1px, 10em, 3rem, 2ch);
}
```
