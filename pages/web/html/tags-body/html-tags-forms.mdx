## `<form>`

Represents a <em>form</em>.

<Badge color="blue">Attributes related to submission</Badge>
<ul>
  <li>
    <Attribute>action=[...]</Attribute> &mdash; the URL to which the form is submitted (which URL
    performs some action on the form data)
  </li>
  <li>
    <Attribute>method=[...]</Attribute> &mdash; the HTTP method with which the form is submitted;
    specify either <code>post</code> or <code>get</code>
  </li>
  <li>
    <Attribute>enctype=[...]</Attribute> &mdash; if <code>method="post"</code>, this specifies the
    MIME type of the form submission. Can be:
    <ul>
      <li>
        <code>application/x-www-form-urlencoded</code> <i>(the default value)</i>
      </li>
      <li>
        <code>multipart/form-data</code>{' '}
        <i>
          (use if there is any <code>{`<input type="file">`}</code>)
        </i>
      </li>
    </ul>
  </li>
  <li>
    <Attribute>novalidate</Attribute> &mdash; don't validate form before being submitted
  </li>
  <li>
    <Attribute>target=[...]</Attribute> &mdash; where to display the response (
    <a href="/web/html/attributes/html-attributes-recurring#target">read here</a>)
  </li>
</ul>

<Badge color="blue">Other attributes</Badge>
<ul>
  <li>
    <Attribute>accept-charset=[...]</Attribute> &mdash; space-separated character encodings the
    server accepts, e.g. <code>utf-8</code>
  </li>
  <li>
    <Attribute>autocomplete=[...]</Attribute> &mdash; indicates whether input fields can be
    autocompleted by the browser; specify either <code>on</code> or <code>off</code>
  </li>
  <li>
    <Attribute>name=[...]</Attribute> &mdash; name of the form (must be unique)
  </li>
  <li>
    <Attribute>rel=[...]</Attribute> &mdash;{' '}
    <a href="/web/html/attributes/html-attributes-recurring#rel">read here</a>
  </li>
</ul>

## `<label>`

Represents the caption of a form field.

It's not a normal text &mdash; if you click on a label, the associated form field will receive focus.

<Badge color="blue">Possible attributes</Badge>
<ul>
  <li>
    <Attribute>for=[...]</Attribute> &mdash; <code>id</code> of a form item to be associated with.
  </li>
</ul>

<WebEditor
  html={smartTrim(`
    <div>
      <input type="checkbox" name="peas" id="peas">
      <label for="peas">I agree with the terms and conditions</label>
    </div>
  `)}
/>

## `<input>`

Creates an interactive input field.

<Badge color="blue">Common attributes (apply to most, if not all)</Badge>
<ul>
  <li>
    <Attribute>name=[...]</Attribute> &mdash; name of this input
  </li>
  <li>
    <Attribute>value=[...]</Attribute> &mdash; value of this input
  </li>
  <li>
    <Attribute>type=[...]</Attribute> &mdash; type of this input, read next section
  </li>
  <li>
    <Attribute>autofocus</Attribute> &mdash; sets focus on input when the page has finished loading
  </li>
  <li>
    <Attribute>disabled</Attribute> &mdash; disables input (the user cannot interact with it)
  </li>
  <li>
    <Attribute>readonly</Attribute> &mdash; makes the user unable to change this input's value
  </li>
  <li>
    <Attribute>required</Attribute> &mdash; forces the user to specify this input's value before
    submitting the form
  </li>
  <li>
    <Attribute>autocomplete=[...]</Attribute> &mdash; hints if the browser can autofill this input;
    specify either <code>on</code> or <code>off</code>
  </li>
  <li>
    <Attribute>form=[...]</Attribute> &mdash; <code>id</code> of a <code>{`<form id="...">`}</code>{' '}
    this element is associated with (
    <a href="/web/html/attributes/html-attributes-recurring#form">read here</a>)
  </li>
  <li>
    <Attribute>list=[...]</Attribute> &mdash; <code>id</code> of a <code>{`<datalist id="...">`}</code>{' '}
    this element is associated with
  </li>
</ul>

## `<output>`

Creates an output container. Its value/content is <em>not</em> submitted during form submission.

Does nothing by itself; you will need to implement the behavior using JavaScript.

<Badge color="blue">Common attributes (apply to most, if not all)</Badge>
<ul>
  <li>
    <Attribute>name=[...]</Attribute> &mdash; name of this output
  </li>
  <li>
    <Attribute>for=[...]</Attribute> &mdash; space-separated <code>id</code>'s of <code>{`<input id="...">`}</code>{' '}
    elements that contribute to the output
  </li>
  <li>
    <Attribute>form=[...]</Attribute> &mdash;{' '}
    <a href="/web/html/attributes/html-attributes-recurring#form">read here</a>
  </li>
</ul>

<WebEditor
  html={smartTrim([
    `
      <!-- TIL: Accesing \`<span id="xxx">\` node can be via \`xxx\` variable in JS,
      although probably it would be better to use \`document.getElementById\` -->
    `,
    `
      <!-- https://developer.mozilla.org/en-US/docs/Web/HTML/Element/output -->
      <form oninput="result.value = parseInt(a.value) + parseInt(b.value)">
        <input type="range" id="b" name="b" value="50"> +
        <input type="number" id="a" name="a" value="10"> =
        <output name="result" for="a b">60</output>
      </form>
    `
  ])}
/>

## `<option>`

Represents an option inside `<select>` or `<datalist>`.

<Badge color="blue">Possible attributes</Badge>
<ul>
  <li>
    <Attribute>value=[...]</Attribute> &mdash; value of the option (to be sent if the option is
    selected). If omitted, the value is taken from the tag's content.
  </li>
  <li>
    <Attribute>label=[...]</Attribute> &mdash; label of the option (the one displayed to the user).
    If omitted, the value is taken from the tag's content.
  </li>
  <li>
    <Attribute>disabled</Attribute> &mdash; makes option unselectable
  </li>
  <li>
    <Attribute>selected</Attribute> &mdash; makes option selected by default
  </li>
</ul>

  <li>
    <Attribute>autofocus</Attribute> &mdash; sets focus on input when the page has finished loading
  </li>
  <li>
    <Attribute>autocomplete=[...]</Attribute> &mdash; hints if the browser can autofill this input;
    specify either <code>on</code> or <code>off</code>
  </li>
  <li>
    <Attribute>spellcheck=[...]</Attribute> &mdash; controls spellcheck; specify either{' '}
    <code>on</code>, <code>off</code>, or <code>default</code>
  </li>
  <li>
    <Attribute>wrap=[...]</Attribute> &mdash; controls line wrap. For both values, the browser
    ensures all line breaks consist of CR+LF pair. Can be:
    <ul>
      <li>
        <code>hard</code>{' '}
        <i>
          (insert line breaks for lines exceeding <code>cols</code>)
        </i>
      </li>
      <li>
        <code>soft</code> <i>(do not insert additional line breaks)</i>
      </li>
    </ul>
  </li>
  <li>
    <Attribute>form=[...]</Attribute> &mdash; <code>id</code> of a <code>{`<form id="...">`}</code>{' '}
    this element is associated with (
    <a href="/web/html/attributes/html-attributes-recurring#form">read here</a>)
  </li>
</ul>

    <Attribute>minlength</Attribute> &mdash; minimum number of characters
  </li>
  <li>
    <Attribute>maxlength</Attribute> &mdash; maximum number of characters

## `<datalist>`

Contains a set of `<option>` elements that represents permissible/recommended options for a field.

You refer to this element by giving this element's `id` in `<input list="...">`.

<WebEditor
  html={smartTrim([
    `
      <!-- input -->
      <label for="ice-cream-choice">Choose a flavor (try double-clicking below):</label><br>
      <input list="flavors" id="ice-cream-choice" name="ice-cream-choice">
    `,
    `
      <!-- data list -->
      <datalist id="flavors">
        <option value="Chocolate"/>
        <option value="Strawberry"/>
        <option value="Vanilla"/>
      </datalist>
    `
  ])}
/>

## `<fieldset>` and `<legend>`

`<fieldset>` groups fields together.

`<legend>` represents the caption of a `<fieldset>`.

<WebEditor
  html={smartTrim(`
    <form>
      <fieldset>
        <!-- legend -->
        <legend>Choose your move</legend>
        <!-- options -->
        <input type="radio" name="move" id="scissors">
        <label for="scissors">Scissors</label>
        <br>
        <input type="radio" name="move" id="rock">
        <label for="rock">Rock</label>
        <br>
        <input type="radio" name="move" id="paper">
        <label for="paper">Paper</label>
      </fieldset>
    </form>
  `)}
/>
