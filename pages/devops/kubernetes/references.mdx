# References

For this module, I mainly use the following reference(s):

- [_Certified Kubernetes Administrator (CKA) Study Guide_](https://learning.oreilly.com/library/view/certified-kubernetes-administrator/9781098107215/) by Benjamin Muschko &mdash; the main reference. The book is available via subscription on learning.oreilly.com.
- [Kubernetes: From Basics to Guru](https://learning.oreilly.com/playlists/330a1112-13ee-4e72-8b2a-6fd8766fddae/) by Sander van Vugt &mdash; as the secondary reference. The playlist is available via subscription on learning.oreilly.com.
- [kubernetes.io](https://kubernetes.io/docs/setup/) &mdash; as the main source of Kubernetes documentation
